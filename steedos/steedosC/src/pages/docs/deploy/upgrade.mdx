</alert>

## Docker环境升级

可以修改 docker-compose 文件中的版本号。

最新发布的版本可以查询 https://hub.docker.com/r/steedos/steedos-community/tags

## 开发环境升级

根目录下有一个package.json文件，该文件中可以维护项目中各个依赖的npm包的版本号。

最新发布的版本可以查询 https://www.npmjs.com/package/@steedos/service-community

### 升级到最新稳定版本

```json
"dependencies": {
  "@steedos/service-community": "^2.2"
},
```
  
### 升级到最新Beta版本

```json
"dependencies": {
  "@steedos/service-community": "next"
},
```

修改好package.json文件中的版本号后，命令行中`cd`到项目目录并执行下`yarn`即可安装相关依赖包，待安装完成后再执行下`yarn start`即可以新版本启动服务。

### 查看当前版本号

点击页面右上角“头像”图标，然后点击“关于”即可查看当前华炎魔方版本号。
